Welcome to grocery Store

Steps to setup the project :
1. In MSSSMS create database and run schema only query then run Data base query.

2. then , Run the backend in visual studio and then run the Front end 

3. For Frontend run "npm install" fro installing all the require packages and then run "ng serve --open" to run the application.

4. Configure both the base urls 'backend into frontend ' AND ' frontend into backend '

5. then run the backend application 

6. Explore the Application.
